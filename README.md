# Portfolio-Webpage
This repository is for my portfolio projects through The Tech Academy

### The basis and features of this webpage

- A simple, text-focused personal portfolio to introduce Garrett Fisher, highlight his education and skills in software development (with a focus on C# and .NET), showcase selected projects, and provide contact options for potential employers or collaborators.
- Target Audience: Recruiters, hiring managers, and anyone interested in Garrett's work as an up-and-coming full-stack developer.
- Introduction and welcome message
- About Me – Personal bio, education, and skills
- GitHub Link – Direct access to code repositories
- Contact – Form or section for reaching out (anchor-linked for quick access)
